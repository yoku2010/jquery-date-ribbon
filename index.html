jQuery Date Ribbon

This Plugin Create Date Ribbon

View project onGitHub

Description

This jQuery plugin creates a date ribbon for your website. Currently this is available only in one theme. Soon i will be launch it in 5 themes more.

Current version: 1.0

Download this plugin and use it to display date in a new style on your website.

Features

  • Fully customized
  • Tested in: jQuery 1.4.3+ in Firefox, Safari, Chrome, Internet Explorer 7, 8, 9, Opera 11
  • Lightweight: 7KB of JavaScript (less than 3KBs gzipped).
  • Load before and load after callback

Instructions & Help

First read usage of plugin (see below). If you found any issue or want any help submit it at GitHub or email me yoku2010@gmail.com

Usage

Date Ribbon accepts settings from an object of key/value pairs (JSON), and can be assigned to any HTML element.

// Format:
$(selector).dateRibbon({key:value, key:value, key:value});

// Examples:
$("#ribbon1").dateRibbon({
			title: "GitHub",
			className: "ribbon",
			href:"https://github.com/yoku2010/jQueryDateRibbon",
			target:"_blank",
			date:new Date()
		});
$("#ribbon2").dateRibbon({
			titleFormat:"days"
			className: "ribbon",
			href:"https://github.com/yoku2010/jQueryDateRibbon",
			target:"_blank",
			date:new Date(2012,8,12)
		});

Settings

Property Default Description
title null This can be use to change title of the date ribbon.
titleFormat days The title Format . Can be set to "days", "weeks", "months", or "years".
className ribbon Class name of date ribbon
href # Date ribbon href location or JavaScript function name
target null href location target can be set to "_blank", "_self","_parent","_top" or "frame-name"
date new Date() Date to show in date ribbon
functionBefore function () { return true; } To fire a function just before date ribbon load. If function return true then create ribbon otherwise no
functionAfter function () { } To fire a function just after date ribbon load.
